# SILVER KISS — Бижута от сребро 925

Кинематографичен едностраничен сайт за марка сребърни бижута, със scroll-reveal hero
анимация, динамични секции и вграден **админ панел** за редакция на цялото съдържание.
Цялата страница се рендира от една централна конфигурация (`content.js`), което я прави
лесна за преизползване като шаблон.

**🔗 На живо**

|---|---|
| Сайт | https://silver-kiss.vercel.app |
| Админ панел | https://silver-kiss.vercel.app/admin · демо парола `silver2026` |

## ✨ Функционалности

- **Scroll-reveal hero** — потъмнялото бижу се „разкрива" при скрол чрез CSS маска;
  гигантското заглавие пада буква по буква. Двупосочно (напред/назад) + Ken Burns „дишане".
- **Динамични секции** — Манифест, Колекции (preview при hover), Каталог с филтри и
  360° завъртане на картите, Материали (брояч-анимации), Акцент, Отзиви (ротация),
  Lookbook (безкрайна лента), Footer с нюзлетър.
- **Изцяло мобилно оптимизиран** — хамбургер меню с навигация до секциите, видимо hero
  заглавие, inline снимки в Колекции, 360° завъртане при скрол на тъч, по-бърз reveal.
- **Админ панел** — редакция на всеки текст, цена, снимка; добавяне/изтриване на продукти
  и отзиви; слайдер за мекотата на reveal ефекта; запис в браузъра + експорт/импорт на JSON.
- **Достъпност** — `aria` етикети, уважава `prefers-reduced-motion`.
- Всички цени в **евро (€)**, целият интерфейс на **български**.

## 🛠 Технологии

- **React 18** + **Babel Standalone** (JSX се компилира в браузъра — без build стъпка)
- **GSAP** + **ScrollTrigger** за анимациите
- Чист **CSS** (`app.css`), Google Fonts: Michroma, Manrope, Cormorant Garamond
- **Vercel** за хостинг (статичен сайт, авто-деплой от `main`)

Без bundler, без `node_modules` — отваря се директно като статични файлове.

## ▶️ Локално стартиране

Понеже JSX се зарежда чрез `fetch`, нужен е прост HTTP сървър (отварянето като
`file://` няма да работи):

```bash
# Python
python -m http.server 8000

# или Node
npx serve .
```

После отвори `http://localhost:8000` (сайт) и `http://localhost:8000/admin.html` (админ).

## ⚙️ Редакция на съдържанието

Цялото съдържание живее в обекта `SK_DEFAULT` в [`content.js`](content.js) — марка,
навигация, hero текстове и изображения, продукти с имена/цени/снимки, материали, отзиви и т.н.

Два начина за промяна:

1. **Чрез админ панела** (`/admin`) — визуално, без код. Промените се пазят в `localStorage`
   на браузъра и могат да се експортират като `content.json`.
2. **Директно в кода** — редактирай `SK_DEFAULT` в `content.js` за постоянни, версионирани
   промени.

> **Бележка:** в момента промените от админа са локални за устройството (демо ниво). За да
> станат постоянни и видими за всички посетители, виж [`BACKEND.md`](BACKEND.md) — стъпките
> за свързване на реален Supabase backend.
